Nimrah Café & Bakery — Website
A modern, responsive website built for Nimrah Café & Bakery, an 85-year-old iconic Irani chai café near Charminar, Hyderabad.
This project brings a traditional local business online with a clean and engaging user experience.

✨ Features
-->Sticky navbar with mobile hamburger menu
-->Hero section with business highlights
-->Interactive menu with category filter
-->Customer testimonials & experience sections
-->Contact with WhatsApp, Call & Google Maps
-->Smooth scroll-based animations
-->Fully mobile responsive design

🎨 Design
Warm cream & chai-brown theme with elegant typography and clean UI.

🛠 Tech Stack
HTML, CSS, Vanilla JavaScript

📈 Impact
Improves online visibility, helps customers explore the menu, and enables direct communication with the café.
